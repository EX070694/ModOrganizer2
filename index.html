Mod Organizer 2
Epic Games, GOG, Steam

Introduction:

This is the Mod Organizer 2 page, crafted to help aid in MO2's setup.

Chapter 1: Setup

  • 1. Make a "Mod Organizer 2" folder in the space you want the files to be located.
  • 2. Download and extract the Mod Organizer 2 file to your folder.
  • 3. Make a folder for modded game (Ex: Fallout4 - Mods & Overwrite)
  • The game files have a download folder, an overwrite folder, a profile folder, and a webcache folder.

Chapter 2: Create New Instance

  • 1. Click File in the top right of MO2.
  • 2. Click Create New Instance.
  • 3. Click Next
  • 4. Create Global Instance
  • 5. Click Browse
  • 6a. C:\Program Files\Epic Games\(Select Game)
  • 6b. C:\Program Files (x86)\GOG Galaxy\Games\(Select Game)
  • 6c. C:\Program Files (x86)\Steam\steamapps\common\(Select Game)
  • 7. Click Next
  • 8. Click Next
  • 9. Click Next
  • 10. Click Finish

Chapter 3: Key

  • Paste the below function to MO2's custom.ini field in the tools folder:
  • [Archive]
  • bInvalidateOlderFiles=1
  • [sResourceDataDirsFinal=

Chapter 4: Separators

  • 1. Click Open List Options
  • 2. Click create separator
  • (Left pane will need to be sorted by priority to use.)

Chapter 5: Join Game and Script Extender to MO2

  • 1. Open the dropdown menu within MO2s Loader area.
  • 2. Click "Edit"
  • 3. Click "Add an Executable".
  • 4. Click "Add from File".
  • 5. Navigate to and select your executable application.
  • 6. Apply your executable and click OK.

Chapter 6: Add Program Shortcut to MO2

  • 1. Select your program within MO2's program loader.
  • 2. Open the shortcut dropdown.
  • 3. Click "Toolbar and Menu".

Chapter 7: Enable Bodyslide to Read Files

  • I was having the issue of the Outift/Body and Presets tab being blank but fixed it by going to MO2 and right clicking BodySlide and Outfit Studio in mods tab then clicking open in explorer. The path shown is the same path you need to set Bodyslide to when editing the executable. Hope this helps anyone having the same problem!

Chapter 8: Update, Replace F4SE

  • 1. Download newest version of F4SE
  • 2. Extract newest F4SE to desktop
  • 3. Drag all of the F4SE folder contents to game directory
  • 4. Set new loader.
